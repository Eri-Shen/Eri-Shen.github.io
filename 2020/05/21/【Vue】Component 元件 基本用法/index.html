Vue Component 元件 基本用法

▌前言

用vue開發網頁就可以想成將網頁拆成好幾個元件，我們在分別去控制元件的，上圖取自vue官方網站，最上面就是root，下方接好幾個不同的元件
我們用vue開法者工具看

可以看到root的下方接了這個名稱的原件，而我們可以分別、獨立的去控制這些元件的html、css、js、資料等等

▌宣告Vue Component

vue也長的很像一開始我們在宣告的vue app，一樣都有data、methods…等等這些屬性
下面是普通vue app宣告

Vue Component的宣告有兩種，分別是全域以及部分宣告

▌全域宣告

全域宣告使用Vue.component()來宣告一個vue元件
()裡面會接兩個參數，第一個是「元件的名稱」，第二個是「相關資料」，為一個物件。template裡面放html，也就是之後在html渲染出的樣子

以下有兩點需要特別注意的地方
:::warning

  1. 命名：元件的命名可以使用kebab-case（中間用－串聯）或是PascalCase（開頭大小寫），但因為到時候我們使用元件的方法是將它變成像html標籤，但是html對大小寫不敏感，所以我都會用kebab-case命名，避免出問題
  2. data不是直接接一個物件，而是要用一個function(){return {}}回傳一個物件
    :::

▌部分宣告

先宣告一個新裡的「相關資料」，接著再到vue app裡建立元件物件，key為元件的名稱，value就是將剛剛宣告的相關資料帶進去

:::info
全域與部分宣告有什麼差別呢?
全域：元件會無條件跟網頁一起被載入，可被所以東西共享
部分：只有在會用到時才會載入，被限制在vue #app裡
:::

▌template與x-template

元件的html程式碼，除了我們將它放在template，還有另外一種x-template，這樣很多一大串的的html程式碼就不用都寫在物件裡，也相對好閱讀
